# SKUSKU Renewal Backend

SKUSKU Renewal은 서경대학교 멋쟁이사자처럼 동아리의 학습 활동을 지원하기 위한 **동아리 웹 플랫폼 백엔드 서버**입니다.

동아리 활동 중 진행되는 교육 세션 이후 학습 내용을 복습할 수 있도록 **복습 퀴즈 시스템**을 제공하며,  
객관식 / 주관식 / 서술형 문제를 통해 학습 내용을 점검할 수 있도록 설계되었습니다.

특히 서술형 문제의 경우 **AI 기반 채점 시스템**을 통해 답안을 분석하여 자동 채점을 진행합니다.

---

# 📌 Project Overview

기존 동아리 홈페이지 기능을 확장하여  
학습 콘텐츠 관리와 복습 퀴즈 기능을 제공하는 **웹 기반 학습 플랫폼**을 구축하는 프로젝트입니다.

교육 세션 이후 학습 내용을 점검하고 복습할 수 있도록  
퀴즈 생성, 응시, 채점, 결과 확인 기능을 제공합니다.

---

# 🚀 주요 기능

### 복습 퀴즈 생성
- 관리자가 교육 내용을 기반으로 퀴즈 생성
- 객관식 / 주관식 / 서술형 문제 유형 지원
- 문제에 **이미지 첨부 기능 제공**

### 퀴즈 이미지 업로드
- 문제 이미지 업로드 시 **AWS S3에 저장**
- 서버에서는 이미지 URL을 관리하여 클라이언트에 전달

### 퀴즈 응시
- 사용자가 게시된 퀴즈에 응시
- 문제 유형에 따라 자동 채점 또는 AI 채점 진행

### 자동 채점
- 객관식 및 단답형 문제는 서버 로직을 통해 자동 채점

### AI 기반 서술형 채점
- OpenAI API를 활용하여 서술형 답안을 분석
- 답변 유사도 및 의미 기반 채점 진행

### 결과 조회
- 퀴즈 응시 결과 확인
- 점수 및 채점 결과 제공
